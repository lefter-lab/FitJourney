Инициализиране и качване на проект в GitHub
Ако проектът няма git все още
1. Инициализирай git в проекта
bash
git init
2. Добави всички файлове
bash
git add .
3. Първи commit
bash
git commit -m "Initial commit"
4. Свържи с GitHub репото
bash
git remote add origin https://github.com/lefter-lab/FitJourney.git
5. Провери дали връзката е добавена
bash
git remote -v
6. Push към GitHub
bash
git push -u origin main
Ако клонът се казва master вместо main:

bash
git push -u origin master
Ако репото в GitHub вече има файлове (README и т.н.)
Трябва първо да pull-неш:

bash
git pull origin main --allow-unrelated-histories
git push -u origin main
Ако проектът вече има git, но няма remote
Пропусни git init и директно:

bash
git remote add origin https://github.com/lefter-lab/FitJourney.git
git push -u origin main
Флагът -u се ползва само първия път — след него просто git push е достатъчно.
